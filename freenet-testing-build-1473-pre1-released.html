Freenet testing build 1473-pre1 released

Sun 10 April 2016
By Freenet Contributors

Freenet unstable testing prerelease build 1473-pre1 is now available. It's the first build to require Java 7.

Highlights:

  • Add The Filtered Index to the default bookmark list
  • Improve MP3 filter stability
  • Fix binary blob download over FCP
  • Fix handling of filenames with non-ASCII spaces

For more details and information on how to test this release see the development mailing list thread.

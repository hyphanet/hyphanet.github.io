Developers

If you already have a contribution in mind you can find the project repositories on GitHub.

  • Get in contact in the mailing lists
  • Join us on IRC: #freenet on irc.libera.chat
  • Report issues in the bug tracker hosted by MantisHub

The Wiki contains a list of freenet-related projects and documentation of the Hyphanet Client Protocol for using Hyphanet from external programs.

We're happy to accept pull requests on GitHub as well as patches sent under a pseudonym through Hyphanet forums such as FMS.
